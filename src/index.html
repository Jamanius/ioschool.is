Overview of our adventure

ioschool is an experiment in open source computer science education. It is based heavily on the existing work of the Node School and Cyber Wizard Institute curiculuums. We're making this up as we go, if you have feedback please let us know at ioschool/discussions.

During our ioschool journey, we will be learning how to harness the magic of signals and pixels to produce art magic and mad science. While there isn't an explicit focus on learning for the sake of being hired in industry, rest assured that if you become proficient at what is being taught, you will ascend to be a higher level wizard with more than enough skills to pay the bills.

While programming can be done in a multitude of languages across many platforms, we will be focusing on javascript (using Node.js) on Unix-based (Linux or Mac OS) and web browser environments.

Gearing up for the journey ahead

To begin, we first need a computer that runs Linux or Mac OS. If you have Windows, we will help you install Linux - but you have to be willing to have Linux!

Then, you need to follow our guide to install Node.js.

Next, you need a Text Editor. If you don't have one already, you may want one for editing code. A few options: Atom Sublime Text, Light Table, TextMate, Brackets.
